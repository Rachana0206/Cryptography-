The Playfair Cipher is a digraphic substitution cipher that uses a 5x5 matrix based on a keyword to encrypt pairs of letters.

üìö Theory & How It Works

The Playfair Cipher encrypts pairs of letters (digraphs) instead of single letters, making it more secure than simple substitution ciphers. It uses a 5x5 matrix filled with the letters of the alphabet (I and J share the same position).

Rules:
1. If both letters are in the same row: take the letter to the right
2. If both letters are in the same column: take the letter below
3. If neither: form a rectangle and take the opposite corners

Example with Keyword "PLAYFAIR":

Keyword: PLAYFAIR

Matrix: P L A Y F, I R B C D, E G H K M, N O Q S T, U V W X Z

Plaintext: HELLO WORLD

Process: HE ‚Üí LL ‚Üí OW ‚Üí OR ‚Üí LD

Security: The Playfair Cipher is more secure than simple substitution ciphers because it encrypts digraphs, making frequency analysis more difficult. However, it can still be broken using known plaintext attacks.
