The Hill Cipher is a polygraphic substitution cipher based on linear algebra that uses matrix multiplication for encryption and decryption.

The Hill Cipher encrypts groups of letters (digraphs for 2x2 matrix, trigraphs for 3x3 matrix) using matrix multiplication. Each letter is converted to a number (A=0, B=1, ..., Z=25), and the text is processed in blocks.

Security: The Hill Cipher is more secure than simple substitution ciphers because it encrypts multiple letters at once, making frequency analysis more difficult. However, it's vulnerable to known plaintext attacks.
