DES is a symmetric-key block cipher that encrypts data in 64-bit blocks using a 56-bit key. It was the standard encryption algorithm from 1977 to 2001.

üìö Theory & How It Works

DES is a block cipher that uses a Feistel network structure. It processes 64-bit blocks of plaintext using a 56-bit key (plus 8 parity bits) through 16 rounds of encryption. Each round applies substitution and permutation operations to provide confusion and diffusion.

‚Ä¢ Block size: 64 bits (8 bytes)
‚Ä¢ Key size: 56 bits (7 bytes) + 8 parity bits
‚Ä¢ Rounds: 16 Feistel network rounds
‚Ä¢ Modes: ECB, CBC, CFB, OFB

Process: Initial Permutation ‚Üí 16 Rounds ‚Üí Final Permutation

Each Round: Right half ‚Üí F-function ‚Üí XOR with left half ‚Üí Swap halves

Output: 64-bit encrypted block

Security: DES was broken in 1997 using brute force attacks. While it's no longer considered secure for modern applications, it's still used in educational contexts and some legacy systems. Triple DES (3DES) was developed to extend its security.
