SHA-256 is a cryptographic hash function that produces a 256-bit (32-byte) hash value, commonly used in digital signatures and blockchain technology.

üìö Theory & How It Works

SHA-256 is a member of the SHA-2 family of cryptographic hash functions. It takes an input of any length and produces a fixed-size 256-bit output. The algorithm is deterministic, meaning the same input will always produce the same output, and it's designed to be collision-resistant.
